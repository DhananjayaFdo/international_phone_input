## [1.0.0] - 2025-02-13

### Added
* Initial release of international_phone_input package
* `InternationalPhoneInput` widget with country selector
* Support for 240+ countries with dial codes
* Automatic phone number validation based on country rules
* Searchable country picker bottom sheet
* Multi-language support for country names (20+ languages)
* Extensive customization options (colors, borders, text styles)
* Form validation integration
* Custom validator support
* Min/max length validation per country
* Flag emoji support
* Comprehensive documentation and examples
* Complete test coverage
* Working example app

### Features
* No external dependencies (except Flutter)
* Null safety support
* Platform-independent
* Accessibility support
* Performance optimized
